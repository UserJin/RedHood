# RED HOOD

## 플랫폼

- PC




## 장르

- 2D
- 플랫포머
- 로그 라이트




## 핵심 재미 요소

랜덤으로 등장하는 아이템을 획득하고 아이템의 시너지를 활성화해 다양한 플레이 경험하기




## ***RedHood***의 차별점

잠시 동안 캐릭터가 강해지는 각성 상태를 이용해 더 강력한 플레이 경험하기




## ***RedHood***의 코어 루프

<aside>

### - KILL
죽는 한 까지 던전에서 몬스터 처치

</aside>

<aside>

### - GET
아이템을 획득해 더 많은 몬스터 처치

</aside>

<aside>

### - GROW
특성을 열어 강력한 스킬 사용

</aside>
